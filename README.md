# Challenge

*Question* : There is a list of challenges. Create a module to select a random challenge for player. 

*Conditions*
> If challenge is not completed a new challenge will be presented after 24 hrs.<br>
> If challenge is completed, a new challenge will be presented after 2 hrs.

*Solution*<br>
This is simple game in which player can jump by hitting 'Space'. Player scores 1 for each jump.<br>
Player is presented with a target score as challenge. If player achieves that score he is presented with a new challenge after 2 hours (from when he completed the challenge).<br>
If player can't complete challenge, he is presented with a new challenge after 24 hours (from when he was presented with the challenge).

*Controls*
> Space : Jump to gain score<br>
> R : Reset game
